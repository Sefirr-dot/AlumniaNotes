# Alumnia Notes - Project Documentation

## 1. Project Name

Alumnia Notes

## 2. Project Overview

Alumnia Notes is an application designed to facilitate the management of students in an educational institution. It will allow managing student data, subjects, grades, and attendance. The system will have an intuitive interface and will be based on a client-server architecture using C#, WPF, and SQL Server.

## 3. Functionalities

### Student Management

*   Register, edit, and delete students.
*   Consult each student's academic history.

### Subject Management

*   Add and modify subjects available in the institution.
*   Assign students to specific subjects.

### Grade Management

*   Record and modify grades by subject and student.
*   Calculate averages and generate reports.

### Attendance Management

*   Mark student attendance in each class.
*   Consult historical attendance records.

### Reports and Statistics

*   Generate academic performance reports.

### Authentication and User Roles

*   Login with different roles (administrator, teacher, student).
*   Access restriction according to user type.

## 4. Technologies

*   C#
*   WPF
*   SQL Server
